Générer un ID Locataire Fonctionnalité Générer un ID Locataire Cette fonctionnalité vous permet de générer un ID/code unique à transférer à votre locataire afin qu'il puisse créer son compte et ainsi participer à la gestion de votre bien. Générer un ID {{ID}} copier Une fois votre code généré appuyez sur "Copier" afin de copier l'entièreté de votre code. Vous pourrez ensuite envoyer ce code à votre locataire pour qu'il aie accès à la plateforme.
